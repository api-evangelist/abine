# AbineThis is a repo for managing machine readable artifacts for the [Abine APIs](https://www.abine.com). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Abine APIs](https://www.abine.com).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://www.abine.com)- [Developer](https://www.abine.com)- [Twitter](https://twitter.com/Abine)- [GitHub](https://github.com/Abine)The [APIs.json](https://github.com/api-evangelist/abine/blob/master/apis.json) index provides links to all the relevant links for the Abine APIs, including the OpenAPI and Postman if they are present.
